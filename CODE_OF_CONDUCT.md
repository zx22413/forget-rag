# Code of Conduct

This project adopts the [Contributor Covenant][cc] version 2.1 as its
Code of Conduct. The full text is the authoritative version; please
read it before contributing.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Summary

In short: be kind, assume good faith, focus on the work. Discussion
should stay technical and constructive. Behavior that targets people
rather than ideas — personal attacks, hostility, dismissiveness —
isn't welcome here.

## Scope

This applies to all project spaces: issues, pull requests, the repo
itself, and any related public channels.

## Reporting

If you experience or witness behavior that violates this code, please
open a private security advisory on GitHub:

**https://github.com/zx22413/forget-rag/security/advisories/new**

(Same channel as security reports — it's private and only the maintainer
sees it. Use the title prefix `[CoC]` so it routes correctly.)

The maintainer will respond within 7 days. Reports are kept confidential
to the extent possible.

## Enforcement

The maintainer may, at their discretion, warn, temporarily restrict, or
permanently ban contributors whose conduct repeatedly or seriously
violates this code. Decisions are final but can be appealed via the
same private channel.
